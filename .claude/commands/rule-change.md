Process a rule change request through the governance protocol.

Follow the process defined in `.claude/rules/05-rule-governance.md`:

1. Read ALL existing rules from `.claude/rules/`
2. Analyze the requested change: $ARGUMENTS
3. Check for logical consistency and conflicts with existing rules
4. Adapt wording for clarity
5. Present BEFORE/AFTER comparison
6. WAIT for user confirmation
7. After confirmation: apply changes
8. Log the change in `99_Roland_Work_Place/CHANGELOG_RULES.md`
