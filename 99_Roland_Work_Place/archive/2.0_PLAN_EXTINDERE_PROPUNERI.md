## 📄 DOCUMENTAȚIE PROIECT — Personal Command Center

### 🎯 Descriere generală
O aplicație web accesibilă de pe orice device (browser web + Android) care centralizează tool-uri, fișiere, AI și integrări externe într-un singur panou de comandă personal.

---

### 🏗️ Arhitectură tehnică

```
[Orice device — browser / Android PWA]
              ↓ HTTPS
    [Frontend — Vercel (gratuit)]
              ↓ API calls
    [Backend Agent — Node.js pe laptop]
              ↓
    ┌─────────────────────────────┐
    │  Fișiere locale (C:\Roly)  │
    │  Google Drive API           │
    │  GitHub API                 │
    │  Gmail API                  │
    │  Claude API (AI)            │
    │  OpenWeather API            │
    └─────────────────────────────┘
```

**Frontend:** Vercel (gratuit, accesibil de oriunde)
**Backend/Agent:** Node.js rulând pe laptopul tău ca serviciu Windows
**Bază de date:** SQLite local pe laptop (notițe, tag-uri, setări)
**Mobile:** PWA — instalabil pe Android ca aplicație nativă

⚠️ Fișierele locale sunt accesibile **doar când laptopul e pornit și online.**

#### **M2 — Comenzi AI pe Documente** *(prioritate 2)*

Alimentat de **Claude API (Haiku)** — cost foarte mic per utilizare.

- Rezumă orice document (PDF, Word, TXT)
- Răspunde la întrebări despre conținutul unui fișier
- Generează raport automat dintr-un folder
- Redenumește fișiere inteligent după conținut
- Extrage date structurate din documente (tabele, liste)
- Traduce un document întreg
- Compară două documente și listează diferențele
- Generează un index al tuturor documentelor dintr-un folder

#### **M3 — Integrări Externe** *(prioritate 3)*

**Google Drive:**
- Navighezi structura Drive
- Upload / Download fișiere
- Căutare în Drive
- Preview documente Google Docs

**Gmail:**
- Inbox sumar (ultimele 20 emailuri)
- Citești emailuri
- Trimiți emailuri simple
- Căutare în inbox

**GitHub:**
- Lista repo-urilor tale
- Ultimele commit-uri
- Status branch-uri
- Deschide/vizualizează issues

**Google Calendar:**
- Agenda zilei/săptămânii
- Adaugi evenimente rapid
- Reminder-e vizibile în dashboard

#### **M4 — File Explorer Local** *(prioritate 4)*

- Navighezi `C:\Users\ALIENWARE\Desktop\Roly` și orice alt path configurat
- Creezi / redenumești / muți / ștergi fișiere și foldere
- Upload fișiere noi din browser
- Download fișiere pe device-ul curent
- Preview: PDF, imagini, TXT, cod sursă
- Tag-uri personalizate pe fișiere
- Favorite — foldere accesate frecvent
- Detectare fișiere duplicate
- Sortare după dată, nume, dimensiune, tip

---

#### **M5 — Căutare Full-Text** *(în File Explorer)*

- Caută un cuvânt în **conținutul** tuturor documentelor (nu doar numele)
- Suportă: PDF, Word, TXT, Excel
- Rezultate cu preview contextual (linia unde apare cuvântul)
- Filtrare după tip fișier, dată, folder

---

#### **M6 — Rapoarte & Statistici** *(prioritate 5)*

- Spațiu ocupat per folder (grafic vizual)
- Fișiere modificate recent (timeline)
- Tipuri de fișiere în colecția ta (pie chart)
- Activitate zilnică (ce ai accesat/modificat)
- Fișiere neutilizate de peste 30/60/90 zile
- Export raport PDF cu un click

---

#### **M7 — Dashboard Principal**

- Widgets configurabile (activezi ce vrei)
- Meteo Nădlac/Arad — afișat permanent
- Agenda zilei (Google Calendar)
- Notițe rapide vizibile
- Fișiere recente
- Shortcut-uri către foldere favorite

---

#### **M8 — Comenzi Rapide (Shortcuts)**

Butoane definite de tine pentru acțiuni repetitive:
- "Creează structură proiect nou" → generează folderele standard automat
- "Arhivează fișierele mai vechi de 6 luni"
- "Raport zilnic" → generează și salvează raportul
- Poți adăuga orice script personalizat

### 🛠️ Stack Tehnic Final

| Componentă | Tehnologie | Motiv |
|---|---|---|
| Frontend | React + Vite + Tailwind | Modern, rapid, cunoscut |
| Mobile | PWA (installable) | Fără App Store, gratuit |
| Backend Agent | Node.js + Express | Rulează pe laptop |
| Serviciu Windows | node-windows | Pornește automat cu laptopul |
| Bază de date | SQLite | Zero configurare, local |
| AI | Claude API (Haiku) | Ieftin, rapid, în română |
| Auth | JWT + bcrypt | Standard, sigur |
| Frontend hosting | Vercel (gratuit) | Deja știi, zero cost |
