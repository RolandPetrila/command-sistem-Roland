Propuneri noi — ce NU există în v1.1
🔴 Prioritate înaltă (utile zilnic)
P1 — Terminal Web (SSH în browser)
Accesezi un terminal direct din panou — rulezi comenzi pe server, fără să deschizi PuTTY/CMD. Tehnologie: xterm.js + WebSocket. Gratuit.
P2 — Monitorizare uptime platforme proprii
Verifică automat dacă platform-predare.vercel.app și celelalte proiecte ale tale sunt online. Alertă instant dacă pică. Tehnologie: httpx async ping + WebSocket notificare. Gratuit.
P3 — Pomodoro structurat per proiect
Nu doar cronometru simplu (deja în Faza 11) — ci sesiuni de lucru 25/5 min legate de un proiect specific, cu istoric „câte sesiuni am lucrat la proiectul X azi". Gratuit.
P4 — Voice Notes + Transcriere
Înregistrezi un mesaj vocal din browser → transcriere automată text → salvat în notepad. Util când ai mâinile ocupate. Tehnologie: Web Speech API (gratuit, în browser) + Whisper API (opțional, ~0.006$/minut).

🟡 Prioritate medie
P5 — Integrare n8n
Văd că ai n8n în bookmarks. Panoul poate declanșa workflow-uri n8n cu un buton — ex: „trimite raport săptămânal pe email" sau „sincronizează fișiere cu Drive". Tehnologie: webhook n8n + buton în panou. Gratuit.
P6 — Partajare temporară fișiere (link unic)
Generezi un link cu expirare (ex: 24h) pentru un fișier de pe server — îl trimiți unui client fără să îi dai acces la tot panoul. Tehnologie: token unic SQLite + endpoint FastAPI public. Gratuit.
P7 — Scanner facturi primite (OCR → evidență automată)
Fotografiezi o factură primită → OCR → Claude extrage: furnizor, sumă, dată, număr factură → se adaugă automat în evidența din Faza 10. Completează ciclul: nu doar emiți facturi, ci și înregistrezi ce ai primit. Cost: ~0.01$/factură.
P8 — Mind Map / Schiță rapidă
Tablă albă digitală pentru schițe, diagrame, idei vizuale. Salvat ca imagine/SVG. Tehnologie: tldraw (open source, embed simplu). Gratuit.

🟢 Prioritate normală
P9 — API Tester simplu (Postman-like)
Testezi endpoint-urile propriilor proiecte (platform-predare, casino, etc.) direct din panou — fără să deschizi Postman. Util în dezvoltare cu Claude Code. Gratuit.
P10 — Alerte preț valutar personalizate (extinde 11.5)
Nu doar alertă când EUR/RON trece un prag — ci și alerte pentru mai multe perechi valutare simultan, cu istoric grafic pe 30 zile. Extensie directă a ce e deja planificat.
P11 — Generator coduri de bare
Lângă QR Generator (Faza 11.4) — generezi și coduri de bare EAN-13/Code128. Util pentru activitatea cu Mosslein SRL. Tehnologie: python-barcode. Gratuit.
P12 — Colecție snippets de cod
Salvezi fragmente de cod des folosite (Claude Code prompts, scripts Python, comenzi terminal) cu syntax highlighting și căutare rapidă. Diferit de Notepad — specific pentru cod. Gratuit.
