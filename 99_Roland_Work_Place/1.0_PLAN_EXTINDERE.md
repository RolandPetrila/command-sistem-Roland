## Note tehnice

- **Stack existent**: FastAPI + React + Tailwind + SQLite — se pastreaza pentru toate extensiile
- **API-uri externe necesare**: OpenWeatherMap (gratuit), BNR (gratuit), Claude API (platit per token)
- **Stocare**: SQLite pentru date structurate, fisiere locale pentru documente
- **Deployment**: Vercel (frontend gratuit) + Railway/Render (backend, tier gratuit disponibil)
- **Autentificare**: JWT simplu, single-user — suficient pentru uz personal
- **Toate functionalitatile** pot fi implementate cu Claude Code in VS Code, pas cu pas
