**⚠️ CRITICAL-1 — PWA necesită HTTPS:** Service workers funcționează DOAR pe HTTPS sau localhost. Tailscale dă IP-uri 100.x.y.z (nu localhost) → fără HTTPS, PWA NU se instalează pe Android. Soluție: `tailscale cert <machine>.ts.net` generează certificate TLS valide → uvicorn cu SSL.

**⚠️ CRITICAL-2 — URL-uri hardcoded:** `frontend/src/api/client.js` are `http://localhost:8000` hardcoded (liniile 3, 121) → din Android via Tailscale, browserul încearcă localhost-ul TELEFONULUI → conexiune eșuată 100%. Fix obligatoriu în 8.0: `window.location.origin` + WebSocket dinamic `${protocol === 'https:' ? 'wss:' : 'ws:'}//${window.location.host}/ws/...`

**Estimare:** 3-4 sesiuni Claude Code (revizuit — HTTPS/TLS + URL refactor + PWA test Android)
**Dependințe:** Cont GitHub (gratuit), cont Tailscale (gratuit)
**ELIMINAT față de versiunea anterioară:** Vercel, Railway, CI/CD, JWT auth, domeniu custom, variabile de mediu Vercel/Railway
