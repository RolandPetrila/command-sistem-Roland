**Scop:** Platforma devine accesibilă de pe orice device prin Tailscale, cu vault securizat.
**Prioritate:** URGENTĂ — fără asta, restul nu are sens.

| # | Task | Tehnologie | Cost |
|---|------|-----------|------|
| **8.0** | **⚠️ Refactorizare modulară OBLIGATORIE** — vezi Wave 0 | Module auto-discovery, `backend/modules/` + `frontend/src/modules/`, sidebar dinamic, migrare DB, activity log → SQLite, fix URL hardcoded | Gratuit |
| 8.1 | Inițializare repository GitHub | git + .gitignore complet | Gratuit |
| 8.2 | Tailscale setup + HTTPS/TLS | Tailscale CLI + MagicDNS + `tailscale cert <machine>.ts.net` + uvicorn `--ssl-keyfile` + `--ssl-certfile` + CORS dinamic | Gratuit |
| 8.3 | PWA Android installabil | vite-plugin-pwa + manifest.json + service worker (necesită HTTPS!) | Gratuit |
| 8.4 | Mod offline parțial | Service worker cache pt tool-uri statice (necesită PWA!) | Gratuit |
| 8.5 | API Key Vault (scope redus: doar chei API aplicație) | cryptography (Fernet) + master password → stochează DeepL key, Google OAuth token, GitHub token | Gratuit |

**⚠️ CRITICAL-1 — PWA necesită HTTPS:** Service workers funcționează DOAR pe HTTPS sau localhost. Tailscale dă IP-uri 100.x.y.z (nu localhost) → fără HTTPS, PWA NU se instalează pe Android. Soluție: `tailscale cert <machine>.ts.net` generează certificate TLS valide → uvicorn cu SSL.

**⚠️ CRITICAL-2 — URL-uri hardcoded:** `frontend/src/api/client.js` are `http://localhost:8000` hardcoded (liniile 3, 121) → din Android via Tailscale, browserul încearcă localhost-ul TELEFONULUI → conexiune eșuată 100%. Fix obligatoriu în 8.0: `window.location.origin` + WebSocket dinamic `${protocol === 'https:' ? 'wss:' : 'ws:'}//${window.location.host}/ws/...`

**Estimare:** 3-4 sesiuni Claude Code (revizuit — HTTPS/TLS + URL refactor + PWA test Android)
**Dependințe:** Cont GitHub (gratuit), cont Tailscale (gratuit)
**ELIMINAT față de versiunea anterioară:** Vercel, Railway, CI/CD, JWT auth, domeniu custom, variabile de mediu Vercel/Railway

## NOTE ARHITECTURALE

1. **Stack păstrat:** FastAPI (Python 3.13) + React 18 + Vite + Tailwind + SQLite — fără rescriere
2. **Structură modulară:** `backend/modules/[modul]/` + `frontend/src/modules/[modul]/` — migrare treptată
3. **Baza de date:** SQLite unic (`calculator.db`) cu tabele per modul — baza de date COMUNĂ, nu per modul
4. **API Gateway:** Toate modulele expun endpoint-uri sub `/api/` — viitoare proiecte se pot conecta
5. **WebSocket:** Deja implementat pentru progress — extins la notificări real-time
6. **Background tasks:** FastAPI BackgroundTasks pentru indexare, OCR batch, sync Drive
7. **Securitate:** Tailscale (rețea privată, fără auth JWT) + API Key Vault cu master password
8. **PWA:** Service worker cachează tool-urile statice — calculator, notepad, QR funcționează și offline
9. **Self-healing (16.10):** Dacă Health Monitor detectează eroare cunoscută → încearcă fix automat
10. **Migrare DB:** Sistem SQL numerotate (`migrations/001_*.sql`) + tabelă `schema_version` — NU alembic (ar necesita SQLAlchemy)
11. **Sidebar dinamic:** Categorii colapsibile, alimentat din module registry — nu static 6 items
12. **Activity log → SQLite:** Migrare de la `activity_log.json` (fișier, max 1000) la tabelă SQLite — scalabilitate + query-uri
